---
title: login
description: API reference for the `turbo login` command.
---

Log in to your Remote Cache provider.

The default provider is [Vercel](https://vercel.com/). To specify a different provider, use the `--url` and `--api` options.

## Flag options

### --api \<url>

The API URL of the Remote Cache provider.

```bash
turbo login --api=https://acme.com/api
```

### --url \<url>

The URL of the Remote Cache provider.

```bash
turbo login --url=https://acme.com
```

### --sso-team \<team>

Connect to an SSO-enabled team by providing your team slug.

```bash
turbo login --sso-team=slug-for-team
```
