<Steps>
<Step>
## Install Turborepo

Install `turbo` globally so you can conveniently run `turbo` commands from anywhere in your repository.

<Tabs items={["npm", "yarn", "pnpm"]} storageKey="selected-pkg-manager">
  <Tab value="npm">
  ```bash
  npm install turbo --global
  ```

  </Tab>
  <Tab value="yarn">
  ```bash
  yarn global add turbo
  ```

  </Tab>
  <Tab value="pnpm">
  ```bash
  pnpm install turbo --global
  ```

  </Tab>
</Tabs>

</Step>
