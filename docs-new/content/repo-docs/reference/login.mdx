---
title: login
description: API reference for the `turbo login` command.
---

Log in to your Remote Cache provider.

The default provider is [Vercel](https://vercel.com/). To specify a different provider, use the `--url` and `--api` options.

## Flag Options

### --url

The URL of the Remote Cache provider. Defaults to `https://vercel.com`.

### --api

The API URL of the Remote Cache provider. Defaults to `https://vercel.com/api`.

### --sso-team

Connect to an SSO-enabled Vercel team by providing your Team slug. (`turbo login --sso-team=slug-for-team`)

| Option     | Type   | Description                                                                                                 |
| ---------- | ------ | ----------------------------------------------------------------------------------------------------------- |
| --url      | string | Defaults to `https://vercel.com`                                                                            |
| --api      | string | Defaults to `https://vercel.com/api`                                                                        |
| --sso-team | string | Connect to an SSO-enabled Vercel team by providing your Team slug. (`turbo login --sso-team=slug-for-team`) |
