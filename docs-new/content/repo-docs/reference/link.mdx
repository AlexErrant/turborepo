---
title: link
description: API reference for the `turbo link` command.
---

Link the repository to Remote Cache. The selected owner (either a user or an organization) will be able to share [cache artifacts](/repo/docs/core-concepts/caching) through [Remote Caching](/repo/docs/core-concepts/remote-caching).

| Option | Type   | Default                  |
| ------ | ------ | ------------------------ |
| --api  | string | `https://api.vercel.com` |
