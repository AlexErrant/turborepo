Extend your Turborepo with new apps and packages.

- [`turbo generate run [generator-name]`](#run-generator-name): Run custom generators defined in your repository.
- [`turbo generate workspace [options]`](#workspace): Create a new packages in your repository by copy an existing one or starting from scratch.

The default command for `generate` is `run` so `turbo generate` is the same as `turbo generate run`.

## `workspace`

```sh
turbo gen workspace [options]
```

Create new empty workspaces, copy existing workspaces, add workspaces from remote sources or run custom generators defined using Plop configurations.

| Option                    | Type    | Description                                                                                                                                                                                                                           |
| ------------------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--run`                   | string  | The name for the new workspace                                                                                                                                                                                                        |
| `--name`                  | string  | The name for the new workspace                                                                                                                                                                                                        |
| `--empty`                 | boolean | Defaults to `true`                                                                                                                                                                                                                    |
| `--copy [source]`         | string  | Name of local workspace within your monorepo or a fully qualified GitHub URL with any branch and/or subdirectory.                                                                                                                     |
| `--destination`           | string  | Where the new workspace should be created                                                                                                                                                                                             |
| `--type`                  | string  | The type of workspace to create (`app` or `package`)                                                                                                                                                                                  |
| `--root`                  | string  | The root of your repository (default: directory with _root_ `turbo.json`)                                                                                                                                                             |
| `--show-all-dependencies` | boolean | Prevent filtering dependencies by workspace type when selecting dependencies to add                                                                                                                                                   |
| `--example-path / -p`     | boolean | In a rare case, your GitHub URL might contain a branch name with a slash (e.g. `bug/fix-1`) and the path to the example (e.g. `foo/bar`). In this case, you must specify the path to the example separately: `--example-path foo/bar` |

## `run [generator-name]`

```bash
turbo gen run [generator-name]
```

Run custom generators defined in your repository.

| Option     | Type    | Description                                                             |
| ---------- | ------- | ----------------------------------------------------------------------- |
| `--config` | string  | Generator configuration file (Default: `turbo/generators/config.js`)    |
| `--root`   | string  | The root of your repository (Default: directory with root `turbo.json`) |
| `--args`   | boolean | Prompt answers passed directly to generator                             |
